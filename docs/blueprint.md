# **App Name**: Talk2Me

## Core Features:

- User Authentication: Secure sign-up/sign-in using Firebase Authentication.
- Channel Management: Create and manage public/private channels with real-time updates.
- Real-time Messaging: Send and receive messages in real-time within channels and direct messages using Firestore.
- Direct Messaging: Private one-to-one conversations between users.
- File Sharing: Share files and images with other users, leveraging Firebase Storage.
- Real-time Presence: Display user online/offline status.
- Smart Reply Suggestions: AI-powered tool suggesting contextually relevant replies based on message history in conversations using a LLM.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and calming feel.
- Background color: Light blue (#E8EAF6), a very desaturated variant of the primary color, creating a calm, uncluttered backdrop that makes the primary color stand out.
- Accent color: Purple (#7E57C2), an analogous color, offering visual contrast for highlights and CTAs.
- Body font: 'PT Sans', a clean sans-serif for readability in the chat interface.
- Headline font: 'Playfair', an elegant serif for headings to provide a touch of sophistication. For longer text, use PT Sans.
- Minimalist line icons for channel actions and user status indicators.
- Smooth transitions for message sending and receiving, providing a fluid user experience.
